# 🛠️ Mantenimiento de Electrodomésticos

Sistema de escritorio para la gestión de servicios de mantenimiento y reparación de electrodomésticos, desarrollado como proyecto académico.

## 🚀 Tecnologías
- **C#** (Windows Forms)
- Visual Studio

## 📌 Sobre el proyecto
Aplicación de escritorio (WinForms) que permite administrar el proceso de mantenimiento de electrodomésticos, gestionando la información de clientes, equipos y órdenes de servicio a través de una interfaz gráfica.

# ⚽ Golazo Web

Proyecto web con temática de fútbol, desarrollado para el curso de **Interacción Humano-Computador (IHC)**, enfocado en aplicar principios de usabilidad y diseño de experiencia de usuario (UX/UI).

## 🚀 Tecnologías
- **HTML**
- **CSS**
- **JavaScript**

## 📌 Sobre el proyecto
Landing page interactiva que pone en práctica conceptos de diseño centrado en el usuario, maquetación web y estilos visuales atractivos, en el contexto de una temática deportiva/futbolística.
